# ⛪ Online Christian Fellowship Platform

A web-based platform designed to connect believers globally, facilitating virtual fellowship, small group discussions, prayer requests, and event coordination.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Backend:** PHP
* **Database:** MySQL

## ✨ Features
* **User Authentication:** Secure registration and login for fellowship members.
* **Prayer Wall:** A dynamic space where users can post prayer requests and interact/support others.
* **Virtual Small Groups:** Directory of active online cell groups with meeting links.
* **Event Calendar:** Schedule of upcoming livestreams, Bible studies, and conferences.

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/yourusername/christian-fellowship.git`
2. Import the `database.sql` file into your local MySQL server.
3. Configure your database connection credentials in `config.php`.
4. Run the project using a local server like XAMPP or WampServer.
