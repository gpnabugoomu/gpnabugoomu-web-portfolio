# ⛪ Gideon's Church Website

A sleek, highly functional website built for a modern local church to engage with regular attendees and welcome newcomers.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript
* **Backend:** PHP
* **Database:** MySQL

## ✨ Features
* **Sermon Archive:** Media library filtering sermons by date, speaker, or topic.
* **Ministries Directory:** Dedicated pages for Kids, Youth, and Adult ministries.
* **Connect Cards:** Digital digital visitor cards processed via a PHP backend.
* **Location & Times:** Clear, interactive section detailing service times and integration with maps.

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/yourusername/local-church-site.git`
2. Setup the MySQL database using the provided schema in `/db/church_db.sql`.
3. Update server paths in the project configuration files.
