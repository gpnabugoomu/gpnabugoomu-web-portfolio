# 🎙️ Gideon Speaks - Simple Podcast Hub

A minimalist audio streaming platform where users can browse, search, and listen to podcast episodes directly from their web browser.

## 🛠️ Tech Stack
* **Frontend:** HTML5 (Audio API), CSS3, JavaScript
* **Backend:** PHP (Admin panel to upload new audio tracks)
* **Database:** MySQL (Stores episode metadata, titles, and descriptions)

## ✨ Features
* **Custom Audio Player:** Seamless custom HTML5 audio player controls (Play, pause, skip, volume slider).
* **Episode Categorization:** Organize episodes by seasons, series, or tags.
* **Search Functionality:** Instant client-side or server-side filtering of episodes.
* **RSS Feed Generation:** Built-in PHP script to generate valid RSS for Apple Podcasts/Spotify compatibility.

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/yourusername/simple-podcast.git`
2. Place your audio files inside the `/audio/` directory.
3. Configure your local environment and access the dashboard at `/admin` to upload your first episode.
