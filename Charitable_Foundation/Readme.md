# 🤝 Gideon Cares Foundation

A modern, transparent website for a non-profit organization to showcase ongoing campaigns, share impact stories, and securely collect donor information.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** PHP (For contact forms and newsletter subscriptions)
* **Database:** MySQL (To manage campaign data and donor registries)

## ✨ Features
* **Active Campaigns:** Dynamic display of current fundraising goals and progress bars.
* **Impact Stories:** A blog-style section detailing how donations are changing lives.
* **Volunteer Portal:** A simple form interface for onboarding new volunteers.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop browsing.

## 🚀 Getting Started
1. Clone the repository: `git clone https://github.com/yourusername/charitable-foundation.git`
2. Start your Apache and MySQL services (via XAMPP/MAMP).
3. Move the project folder to your `htdocs` directory.
4. Open `http://localhost/charitable-foundation` in your browser.
